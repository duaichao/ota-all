站长管理使用说明

站长管理使用说明主要介绍站长维护其下属的不同地区供应商和旅行社信息、审核退款和处理纠纷、对其定期回访并收集意见等。

站长管理主要有四大工作模块：站群管理、优惠活动、公司管理、审核退单和处理掉单。

站群管理：站群管理是指站长维护旅行社和供应商信息，定期对其进行回访并收集意见等。站长可以设置站个性化，给下属的不同地区网站首页配置客服、绑定银行卡方便对账管理、管理在线旅行社和供应商。

优惠活动：优惠活动包括设置优惠和套餐，以及产品搭配优惠活动.站长可以根据实际情况设置系统优惠活动，吸引买家在线支付。

公司管理：公司管理下设部门管理和员工管理,层级管理有助于细化员工权限,方便站长监督和管理。

审核退单和处理掉单：旅行社申请退款和因系统原因造成的掉单，站长都应及时处理。

站群管理主要包括站长管理、旅行社管理和供应商管理。

当旅行社长时间未使用账户登录，或者旅行社与供应商等发生业务纠纷时，此时需要启用账户禁用功能(禁用后的旅行社账户下属的所有员工子账户都不能登录)

已经禁用的账户，设置为启用，具体步骤同上。

设置为独立运营的旅行社，供应商可以指定发布产品给它。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“旅行社”，选中所需的旅行社账户，点击修改，分配角色选择“独立旅行社/独立旅行社+充值退款”，核对无误，点击保存，点击独立运营，如下图所示：


1.2.7 设置银行卡号

普通旅行社设置银行卡号，站长会把实际退款金额转入该银行卡。

独立旅行社设置银行卡号，站长会把该旅行社所有的业务金额转入该账户。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“旅行社”，选中所需的旅行社账户，点击银行卡号，填写银行卡基本信息，核对无误，点击确定，如下图所示：


1.2.8 合作商户

与该旅行社的可预订城市相同的所有供应商，作为合作商户的数据范围;可跨站设置合作商户， 订单属于组团社及其所属站，供应商订单不受站限制。

合作商户设置成功后，供应商可以指定发布产品给该旅行社(如独家指定产品)。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“旅行社”，选中所需的旅行社账户，点击合作商户，选择合作的供应商，核对无误，点击确定，如下图所示：


1.2.9 合同

旅行社设置合同权限后，其总部或者门市/分公司的员工在购买产品时必须选择合同号;合同管理有助于总公司监管门市/分公司的业务情况。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“旅行社”，选中所需的旅行社账户，点击合同，系统提示“数据更新成功”，如下图所示：


1.2.10 出发城市

站长控制旅行社可预定出发城市的范围，旅行社可预定多个城市的产品；旅行社可预定城市与合作供应商产品的出发地匹配的指定产品，以及可预定城市与产品的出发地匹配的共享产品。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“旅行社”，选中所需的旅行社账户，点击出发城市，输入并选择出发城市（可设置多个出发城市），点击确定，系统提示“保存出发城市成功”，如下图所示：

1.3.6 授权

站长可以根据供应商的主营业务，分配供应商账户的不同使用权限，使供应商更加多样化，满足不同客户的需求。

1.3.6.1 授权单卖交通

根据供应商的主营业务，站长可授权供应商账户单独售卖交通产品。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“供应商”，选中需要授权的供应商账户，点击授权 - 单卖交通，如下图所示：


1.3.6.2 授权单卖地接

根据供应商的主营业务，站长可授权供应商单卖地接，即不搭配往返大交通的地接产品。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“供应商”，选中需要授权的供应商账户，点击授权 - 单卖地接，如下图所示：


1.3.6.3 授权线路共享

授权线路共享后，供应商可以将产品共享到平台上，供非协议组团社购买，扩大产品的销售渠道。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“供应商”，选中需要授权的供应商账户，点击授权 - 线路共享，如下图所示：


1.3.7 设置银行卡号

供应商设置银行卡号，旅行社购买产品的金额和退款金额会转入该账户。

1.3.8 出发城市

1.3.9 设置专线类型

供应商设置国内/出境类型，方便组团社按供应商类型筛选产品。

登录站长账户，进入站配置页面，选中站长账户，选中所需地区，选中“供应商”，选中所需的供应商账户，点击设置专线类型，选择专线类型，核对无误，点击确定，如下图所示：

3.3 修改密码

修改站长账户密码，分为两种情况:一是登录账户后，直接修改密码;二是忘记登录密码， 此种情况有两种修改密码的方式，一种情况是已绑定手机号码的站长账户，可以通过绑定的手机号码重置密码，另一种情况未绑定手机号码的站长账户，可以联系公司技术人员修改密码。

登录账户后，直接修改密码: 登录站长账户，点击右上角账户名称，点击改密码，输入新密码，确认无误，点击保存，如下图所示：


通过绑定的手机号码重置密码 : 在登录页面点击"找回密码"，输入手机号码和新密码，点击免费获取验证码，系统会给已绑定手机号码发送验证码，输入验证码，确认无误，点击提交，如下图所示：


公司管理员账户忘记密码且没有绑定手机，可以联系所在地区站长修改密码。

登录站长账户， 进入公司管理页面， 选中需要修改密码的公司账户，点击重置公司密码，重置密码成功后，密码恢复为系统初始密码：111111，如下图所示：


站长账户下属的员工账户忘记登陆密码且没有绑定手机，可以联系所在地区站长修改密码。

登陆站长账户，进入公司管理页面，选中站长账户，选中所需部门和需要重置密码的员工账户，点击重置密码，重置密码成功后，密码恢复为系统初始密码：111111，如下图所示：
